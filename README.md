# Youtube-Playlist-Converter

這是一個可以將Youtube播放清單或合輯裡，每部影片的網址提取出來，您可以將輸出的網址全部複製下來，也可以同時開啟所有影片(我知道這功能很廢)，以後可能會增加新功能。
